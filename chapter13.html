Seekho IPC - XIII

Chapter XIII: Offences relating to Weights and Measures (Sections 264 to 267)

Section 264: Fraudulent Use of False Instrument for Weighing

Description

This section addresses the fraudulent use of a false instrument designed for weighing purposes.

Example 1

Using a tampered scale to misrepresent the weight of goods during a transaction.

Example 2

Employing a fake weighing instrument to deceive buyers about the quantity of products sold.

Section 265: Fraudulent Use of False Weight or Measure

Description

This section covers the use of false weights or measures to commit fraud.

Example 1

Using a weight that has been altered to weigh less than its actual weight in transactions.

Example 2

Applying a false measure to reduce the quantity of goods provided to customers.

Section 266: Being in Possession of False Weight or Measure

Description

This section deals with the possession of false weights or measures, knowing them to be false.

Example 1

Owning weights that have been tampered with for the purpose of fraudulent transactions.

Example 2

Keeping false measuring devices with the intent to use them in dishonest trade practices.

Section 267: Making or Selling False Weight or Measure

Description

This section addresses the act of making or selling false weights or measures.

Example 1

Manufacturing false weights with the intent to sell them for fraudulent use.

Example 2

Selling tampered measuring devices to traders for deceitful transactions.
